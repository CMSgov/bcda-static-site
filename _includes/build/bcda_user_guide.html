Temporary Change to Production Requests

Throughout January 2021, users will need to use the `runout` identifier with the /Group endpoint in order to request any claims from BCDA's production environment. Requests to the /Group and /Patient endpoints without the `runout` identifier will result in an error message. When using the `runout` identifier, data returned by the API will be restricted to claims with a service date of 12/31/2020 or earlier.

These changes are in place to ensure that BCDA does not return any data not assigned to your organization during the transition between performance years. Please see the full details in the End of Year Requests section below.
