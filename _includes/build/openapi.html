Generating Client Code

OpenAPI Code Generator

An OpenAPI v3 JSON schema file, which represents the service capabilities of the API, can be accessed through BCDAâ€™s Swagger page.This file can be used to generate client code to interact with BCDA.

There are various tools that can be used to generate client code from an OpenAPI v3 JSON schema file, one of which is Swagger Codegen. It can generate code in dozens of different programming languages and also produces a README describing usage of the newly generated client code.

Throughout, this guide the README files produced when generating code will be invaluable. Consult your libraries README to learn to invoke each API interaction.

Sample GoLang Code

BCDA provides test code written in Golang in our GitHub instance that you can use to set up your system to work with the API. A sample invocation is below:

Golang Sample Code Invocation 

go run bcda_client.go -host=$HOSTNAME -clientID=$CLIENT_ID -clientSecret=$CLIENT_SECRET -endpoint=Patient -resourceType=ExplanationOfBenefit
