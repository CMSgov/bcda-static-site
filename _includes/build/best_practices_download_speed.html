Requesting compressed data

Requesting compressed data can help increase your download speed. You can request compressed data files by specifying the `Accept-Encoding: gzip` header in your download requests. After you finish downloading, uncompress (unzip) the file to access your data.

Sample cURL Command to Download the Data 

curl "https://sandbox.bcda.cms.gov/data/42/afd22dfa-c239-4063-8882-eb2712f9f638.ndjson" \
-H "Authorization: Bearer {access_token}" \
-H "Accept-Encoding: gzip"

Downloading files concurrently

When multiple files are available, you don't need to wait for one download to complete before starting the next. You may make multiple download requests concurrently
